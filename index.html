ReverseAPI Documentation

Clean API reference for injection and user-level utilities.

InjectResource

Executes Lua code inside a target resource context.

Syntax
ReverseAPI.InjectResource(resourceName, luaCode [, injectionType]) → void
Example
ReverseAPI.InjectResource(
  "esx_ambulancejob",
  "TriggerEvent('esx_ambulancejob:revive')"
)

GetUsername

Returns the current user's username.

Syntax
ReverseAPI.GetUsername() → string
Example
local username = ReverseAPI.GetUsername()
print("Username:", username)

GetVehicleControl

Requests network ownership of a vehicle entity. Must be called before performing local vehicle actions.

Syntax
ReverseAPI.GetVehicleControl(vehicle, maxTries) → boolean
Correct Usage
if DoesEntityExist(vehicle) then
    if ReverseAPI.GetVehicleControl(vehicle, 50) then
        -- Safe to interact with the vehicle
    end
end
Example
-- Acquire control of the nearest vehicle and enter it

local ped = PlayerPedId()
local coords = GetEntityCoords(ped)

local vehicle = GetClosestVehicle(
    coords.x, coords.y, coords.z,
    6.0, 0, 71
)

if vehicle ~= 0 and ReverseAPI.GetVehicleControl(vehicle, 50) then
    SetVehicleEngineOn(vehicle, true, true, false)
    TaskEnterVehicle(ped, vehicle, -1, -1, 1.0, 1, 0)
end

SpoofPed

Temporarily overrides the local player ped model. Useful for testing, animations or context simulation.

Syntax
ReverseAPI.SpoofPed(modelHash) → void
Example
-- Temporarily switch ped model

ReverseAPI.SpoofPed(`s_m_m_security_01`)
Wait(1500)

-- Revert to freemode ped
ReverseAPI.SpoofPed(`mp_m_freemode_01`)
